我的第一篇博客文章

​

第一章

环境准备

1.安装node.js 在node.js官网 https://nodejs.org/en/下载对应版本的安装包，再进行安装（可以不选择安装在默认路径）
2.下载git安装，建议将git和node.js安装在同一个文件夹下面，文件夹路径名字建议全英文。
3.Windows操作系统安装完成后需要重启生效（mac需要切换root权限进行查看）；
重启后，win+r或在打开安装软件所在文件夹，或者直接在文件路径处输入cmd并回车打开命令行提示符页面，
输入node -v 查看node.js版本，再输入npm -v 查看npm版本
4.通过淘宝镜像源下载cnpm 通过-g进行全局下载
npm install -g cnpm –registry==https://registry.npm.taobao.org
5.cnmp -v查看版本确认是否安装成功
6.cnpm install -g hexo-cli 全局安装hexo框架
7.在命令行输入hexo init 进行安装hexo （如果是苹果电脑，输入sudo hexo init 进行安装）
同上，hexo -v查看hexo版本，确认是否安装成功

第二章

启动并书写博客

1.建立一个存储博客的英文名称文件夹，用来存放本地博客（建议和第一章安装node目录放一起，名字起为blog）
2.打开cmd切换到此文件夹路径下（具体步骤可参考第一章第三条）;
3.hexo s 启动博客hexo server的缩写，在电脑浏览器中输入localhost：4000 进行访问，看能否跳转到看看本地bolg是否搭建成功
4.hexo n “我的第一篇博客文章” （新建一篇博客文章）
5.在第一步建立的本地博客的对应目录，依次进入source->posts，可以看到”我的第一篇博客文章.md”一个点md文件，直接记事本打开进行编辑或者其他的notepad++、vscode等工具打开进行编辑（需要注意的是，title、data、tags后面一定要加一个空格，否则生成博客时候会报错）
6.hexo clean 清理一下hexo，作用是清除缓存文件 db.json 和已生成的静态文件 public
7.hexo g 生成博客
8.hexo s 打开本地博客端口进行检测，在电脑浏览器中输入localhost：4000 进行访问
